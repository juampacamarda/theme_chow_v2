# Chow Theme

Tema de WordPress personalizado para e-commerce, construido con Bootstrap 5 y WooCommerce.

## Características

- Compatible con WooCommerce
- Diseño responsivo con Bootstrap 5
- Campos personalizados con Secure Custom Fields (SCF)
- Navegación con WP Bootstrap Navwalker
- Soporte para sliders con Owl Carousel

## Instalación

1. Sube el tema a `/wp-content/themes/chow-theme/`
2. Activa el tema en el panel de WordPress.
3. Instala y activa los plugins requeridos:
   - WooCommerce
   - Secure Custom Fields (SCF)
4. Importa los grupos de campos desde `acf-json/` en SCF.
5. Configura las opciones del tema en "Opciones Chow theme".

## Configuración

- **Footer**: Configura dirección, teléfonos, email en SCF > Footer.
- **Slide Home**: Agrega imágenes y enlaces para el slider en SCF > Slide.
- **Formulario Productos**: Configura el formulario de consulta en SCF > Formulario Productos.

## Desarrollo

- Requiere WordPress 5.0+
- PHP 7.4+
- Bootstrap 5.3.0
- jQuery 3.7.1
